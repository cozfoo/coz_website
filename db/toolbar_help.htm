The tools below the text area enter various html and psuedo-html before and after the selected text in the textarea.

• enters a bullet at the cursor position

B: enters "bold" tags arround selected text to make them appear in bold text

I: enters "italics" tags arround selected text to make them appear italicized

Link: enters "anchor" tags arround selected text to create a link. Options are given to make the link either a link to a page or an email link

For links to other pages:
target="_blank" -- opens in a new window
target="_top" -- opens in same window


List: creates an html "list" with selected lines of text. Lines starting with "###DISC###" will have a circular ("disc") bullet. Other lines may be indented with normal spaces

List Example:

###LIST,START
###DISC###Consumer Products & Retail
      Apparel and Footwear
      Beverage
      Consumer Durables

###DISC###Life Sciences
      Pharmaceuticals
      Biotechnology
      Medical Devices
###LIST,END


Table: creates a tabular layout for selected lines. the lines betwee tne table psuedo-tags entered by this tool must have "cells" separated by the vertical bar (|) character

Table Example:

###TABLE,START
text|text
text|text
###TABLE,END
...Note the 'vertical bar' ( | ) separating the table cells


HTML: puts html psuedo-tags around selected text. This text will be sent directly to the browser and may be any HTML
